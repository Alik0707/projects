# Task Management (Spring Boot Pet Project)

## Description
Task Management is a pet project built with Spring Boot.  
The project is a simple REST API for managing tasks and was created for learning backend development and practicing CRUD operations.

## Features
- Create a task
- Get all tasks
- Get task by id
- Update a task
- Delete a task

## Technologies
- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- Maven
- Hibernate

## Project Structure
The project uses a standard layered architecture:
- Controller layer (REST endpoints)
- Service layer (business logic)
- Repository layer (database access)
- Entity / Model layer (data structure)

Clone repository and run:

mvn spring-boot:run
